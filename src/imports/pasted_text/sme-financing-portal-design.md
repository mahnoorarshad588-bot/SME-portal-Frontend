Design a complete enterprise-grade web portal for the "SME Financing Digital Portal" under the State Bank of Pakistan (SBP).

This is a role-based financial services portal where SMEs apply for financing digitally and participating banks process financing applications.

The objective is to create a highly professional, clean, modern, banking-grade, government-backed portal with exceptional UX, minimal clutter, strong usability, and enterprise-level visual design.

The final output should include:

* Complete UX architecture
* User flows
* High-fidelity UI screens
* Responsive desktop layouts
* Component library
* Design system
* Interactive prototype
* Realistic sample data
* Production-ready design

DO NOT create generic dashboards.
DO NOT add unnecessary widgets.
DO NOT add marketing banners.
DO NOT add decorative graphics.
DO NOT add unnecessary charts.
DO NOT add unnecessary cards.

The design should look like a premium banking SaaS platform similar to modern fintech and government financial systems.
